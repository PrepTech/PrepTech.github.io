Build what you've always dreamed of creating.

Preptech is a club that aims to bring coding and tech
to the Flintridge Prep community, and to provide the resources needed to
bring your ideas to life.

Start off with the basics, or dive right in.

Coding Fundamentals

For those of you who are complete beginners, this is for you. Learn essential concepts of programming such as variables and methods with Javascript of Python.

Mobile Development

Almost everyone has interaction with a mobile device nowadays, and with the power and possibilities of code being able to run anywhere in the palm of your hand, the sky is the limit when it comes to mobile applications. We can help you get started with Android or Objective-C & Swift programming in no time.

Web Development

Whether you have an idea for the next Facebook or just want to make a cool website, you can do it with web development. The flexibility and connectivity of web applications along with the growing list of technologies makes web dev all the more versatile. Our HTML/CSS and jQuery track can help you get started on your web devlopment journey.
